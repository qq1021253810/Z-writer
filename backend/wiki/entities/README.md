# 角色实体目录

此目录用于存放从小说中提取的角色实体页面。

## 命名规范
- 文件名：角色名称.md（如：陆远.md、赵天虎.md）
- 格式：Markdown

## 页面结构
```markdown
# 角色名称

## 基本信息
- 姓名：
- 年龄：
- 身份：
- 背景：

## 性格特征
- 决策风格（激进型、稳健型、谋略型）：
- 情商水平：
- 价值观：

## 核心优势
- 决策能力：
- 资源网络：
- 专业技能：
- 人脉关系：

## 人物关系
- 合作伙伴：
- 竞争对手：
- 导师：
- 下属：
- 家人：

## 成长弧线
- 初始状态：
- 关键转折：
- 最终状态：
```

## 使用说明
1. 每创建一个新角色，在此目录下创建对应的 md 文件
2. 随着剧情发展，及时更新角色状态
3. 使用 `/wiki-health` 命令检查角色页面的完整性

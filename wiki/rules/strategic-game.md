# 战略博弈规则

## 核心原则

### 1. 战略博弈深度
- 角色决策基于信息差和利益考量
- 非情绪冲动驱动
- 每个决策都有代价和后果

### 2. 多线并行张力
- 张力来自角色动机冲突
- 非外部强加的紧张感
- 多线交织，相互影响

### 3. 主题深度
- 通过故事探讨深刻主题
- 非简单爽感刺激
- 思想碰撞，价值观冲突

## 战略博弈要素

### 1. 信息差
- 各方掌握的信息不对称
- 信息差是推动博弈的关键
- 信息获取和泄露都是博弈手段

### 2. 利益考量
- 每个角色都有自己的利益诉求
- 利益冲突是博弈的核心
- 短期利益 vs 长期利益

### 3. 权力结构
- 各方力量的对比和制衡
- 弱势方的制衡手段
- 权力格局的变化

### 4. 策略选择
- 进攻策略：主动出击，抢占先机
- 防守策略：稳固阵地，等待时机
- 联盟策略：联合弱者，对抗强者
- 离间策略：分化敌人，各个击破

## 博弈设计模板

### 博弈场景
```
参与方:
  - A方: [目标、资源、弱点]
  - B方: [目标、资源、弱点]
  
信息差:
  - A知道什么？B不知道？
  - B知道什么？A不知道？
  
利益冲突:
  - 核心冲突点是什么？
  - 是否有合作空间？
  
策略选择:
  - A的可能策略？
  - B的可能策略？
  
博弈过程:
  - 第一步：[行动]
  - 第二步：[反应]
  - ...
  
结果:
  - 短期结果？
  - 长期影响？
```

## 禁止事项

- ❌ 无脑碾压（必须有博弈过程）
- ❌ 情绪化决策（必须有理性考量）
- ❌ 单方面优势（必须有制衡）
- ❌ 简单爽感（必须有深度）

## 检查清单

- [ ] 博弈各方是否有明确目标？
- [ ] 信息差是否清晰？
- [ ] 利益冲突是否明确？
- [ ] 策略选择是否合理？
- [ ] 博弈过程是否有张力？
- [ ] 结果是否有代价和后果？

## 示例

### 好的博弈设计
```
场景: 商业谈判

A方（创业者）:
- 目标: 获得投资，保留控制权
- 资源: 核心技术，市场前景
- 弱点: 资金紧张，时间压力

B方（投资人）:
- 目标: 获得高回报，控制公司
- 资源: 资金，行业资源
- 弱点: 错过这个项目的机会成本

信息差:
- A知道技术的真正价值
- B知道行业的竞争格局

博弈过程:
1. A展示技术优势，提高估值
2. B指出市场风险，压低估值
3. A引入其他投资人作为筹码
4. B让步，但要求对赌协议
5. A接受，但设置保护条款

结果:
- 短期: 达成投资协议
- 长期: 双方利益绑定，但也有潜在冲突
```

### 坏的博弈设计
```
问题: 主角轻松碾压对手
缺陷:
  - 没有信息差
  - 没有利益冲突
  - 没有策略博弈
  - 没有代价和后果
```

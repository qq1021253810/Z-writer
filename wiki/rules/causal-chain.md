# 因果链规则

## 核心原则

### 1. 因果链驱动
每个剧情转折必须有前置铺垫，禁止巧合推动。

```
原因 → 结果 → 新原因 → 新结果
```

### 2. 禁止机械降神
- 所有解决方案必须基于已有设定
- 禁止突然出现未铺垫的能力/资源
- 禁止巧合解决关键问题

### 3. 伏笔必须回收
- 埋设的伏笔必须在后续回收
- 回收方式应与埋设时呼应
- 避免悬而未决的伏笔

## 因果链类型

### 1. 直接因果
```
A 发生 → 直接导致 B
```
示例：主角被背叛 → 对他人产生不信任

### 2. 连锁反应
```
A → B → C → D
```
示例：决策失误 → 资源损失 → 地位下降 → 新的危机

### 3. 多因一果
```
A + B + C → 结果
```
示例：信息差 + 利益冲突 + 性格缺陷 → 关系破裂

### 4. 一因多果
```
A → B + C + D
```
示例：关键决策 → 获得资源 + 树敌 + 改变关系格局

## 因果链检查

### 剧情转折检查
- [ ] 转折是否有前置原因？
- [ ] 原因是否在之前铺垫？
- [ ] 是否依赖巧合？
- [ ] 角色行为是否符合动机？

### 解决方案检查
- [ ] 解决方案是否基于已有设定？
- [ ] 是否突然出现未铺垫的能力？
- [ ] 代价是否合理？
- [ ] 后果是否考虑？

### 伏笔检查
- [ ] 伏笔是否清晰埋设？
- [ ] 回收计划是否明确？
- [ ] 回收方式是否呼应？
- [ ] 是否有悬而未决的伏笔？

## 禁止事项

- ❌ 巧合推动关键剧情
- ❌ 机械降神解决问题
- ❌ 未铺垫的能力/资源突然出现
- ❌ 伏笔埋设后不回收
- ❌ 因果链断裂

## 因果链模板

### 事件因果链
```
事件: [描述]
原因:
  - 直接原因: [什么直接导致这个事件？]
  - 深层原因: [什么根本原因导致？]
  - 触发条件: [什么条件触发了这个事件？]

结果:
  - 直接结果: [这个事件直接导致什么？]
  - 连锁反应: [会引发什么后续？]
  - 长期影响: [对整体有什么影响？]
```

### 角色行为因果链
```
行为: [描述]
动机:
  - 表面目标: [角色认为自己在追求什么？]
  - 深层动机: [真正驱动的是什么？]
  
经历基础:
  - 哪些经历导致这个动机？
  
信息基础:
  - 角色掌握什么信息？
  - 信息是否充分？

后果:
  - 行为带来什么后果？
  - 后果如何影响后续？
```

## 示例

### 好的因果链
```
埋设: 主角发现导师的秘密文件（第3章）
铺垫: 导师行为异常、文件内容暗示（第5-8章）
回收: 主角发现导师是幕后黑手（第12章）
```

### 坏的因果链
```
问题: 主角突然获得神秘力量击败敌人
缺陷: 
  - 力量来源未铺垫
  - 获得过程靠巧合
  - 没有代价和后果
```

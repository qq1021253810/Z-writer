# 角色行为逻辑规则

## 核心原则

### 1. 行为逻辑链完整性
每个角色的行为必须遵循完整的逻辑链：
```
经历 → 认知 → 动机 → 行为 → 后果 → 新经历
```

### 2. 智商情商在线
- 角色决策必须符合其认知水平和信息掌握
- 禁止突然降智推动剧情
- 高智商角色应有合理的决策过程
- 情感反应应符合角色性格和经历

### 3. 动机可追溯
- 每个重要行为都必须有明确的动机
- 动机应能追溯到角色的经历和认知
- 禁止无缘无故的爱恨情仇
- 行为变化应有触发事件

### 4. 信息差驱动
- 角色决策基于其掌握的信息
- 信息差是推动剧情的重要动力
- 角色不应知道他们不该知道的事
- 信息获取应有合理途径

## 角色档案要素

### 基本信息
- 姓名、年龄、身份、外貌
- 外貌应有辨识度，能暗示性格或经历

### 行为逻辑链（核心）
- **核心信念**：角色最深层的价值观
  - 形成于什么经历？
  - 是否会动摇？
- **决策模式**：面对选择的优先考量
  - 利益 / 情感 / 原则？
  - 短期 / 长期？
- **行为偏好**：习惯性反应模式
  - 进攻型 / 防守型 / 观察型？
- **底线与禁区**：绝对不做的事
  - 违反时会怎样？

### 背景故事（因果链）
- **出身环境**：如何塑造初始认知？
- **关键经历**：哪些事件改变了认知？
- **当前处境**：基于以上经历，为何处于这个位置？

### 能力与资源
- **核心能力**：基于经历获得的技能（非天赋开挂）
- **资源网络**：人脉、信息、资金等
- **能力边界**：什么是做不到的？代价是什么？

### 动机系统
- **表面目标**：角色自己认为追求的
- **深层动机**：真正驱动行为的心理需求
- **内在冲突**：表面目标与深层动机的矛盾

### 人际关系
- **重要联系人**：关系基于什么？
- **关系中的博弈**：各取所需？相互制衡？

## 禁止事项

- ❌ 完美角色（必须有弱点和盲区）
- ❌ 突然降智（决策必须合理）
- ❌ 巧合推动（必须有因果关系）
- ❌ 脸谱化（没有纯粹的好人/坏人）
- ❌ 无缘无故的爱恨（必须有逻辑基础）

## 检查清单

- [ ] 角色的核心信念是否明确？
- [ ] 决策模式是否一致？
- [ ] 行为能否追溯到动机？
- [ ] 信息掌握是否合理？
- [ ] 是否有弱点和盲区？
- [ ] 人际关系是否有博弈成分？

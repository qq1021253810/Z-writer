# VOF（病毒-癌基因融合体）

## 类型: 概念/设定

## 描述
Viral-Oncogene Fusion，病毒逆转录整合入癌细胞基因组后形成的融合体，是陆沉能力的核心机制。

## 属性
- **全称**: Viral-Oncogene Fusion
- **形成机制**: Ad-ALK-Vec 病毒逆转录整合入 ALK 融合基因
- **激活通路**: MyoD 基因家族（胚胎期肌细胞生成）
- **闭环机制**: 乳酸 → 启动子 → VOF 表达 → 肌细胞生成
- **副产物**: NGF-V（神经生长调节因子）

## 能力输出
### 身体能力
- 核心肌群厚度 2.3 倍
- 快肌纤维占比 78%
- 骨密度提升 40%
- 基础代谢率 1.8 倍
- 反应时 0.15 秒

### 脑域能力（通过 NGF-V）
- 数学天赋：三维坐标系瞬时构建
- 超级记忆：10 秒 200 个数字
- 模式识别：5 倍常人速度
- 多任务处理：3-4 项并行

## 代价机制
- 必须定期无氧运动产生乳酸
- 停止运动 → 共生体系衰退 → 癌症复发
- 对病毒抑制剂（利巴韦林）敏感
- 对 NMDA 受体拮抗剂敏感

# Ad-ALK-Vec 病毒

## 类型: 物品/科技

## 描述
改造型溶瘤腺病毒，基于人类 5 型腺病毒改造，删除 E1B 基因，插入 ALK 蛋白识别序列和双向调控启动子。

## 属性
- **来源**: 暗影生物非法实验室
- **载体**: 人类 5 型腺病毒
- **靶点**: ALK 融合蛋白（仅感染癌细胞）
- **调控机制**: 四环素调控系统，响应乳酸浓度
- **关键改造**:
  - 删除 E1B 基因（避免正常细胞复制）
  - 插入 ALK 蛋白识别序列
  - 插入 p53 蛋白抑制片段
  - 双向调控启动子

## 作用机制
1. 特异性感染癌细胞（通过 ALK 蛋白识别）
2. 在乳酸环境中激活复制
3. 发生逆转录整合，形成 VOF（病毒-癌基因融合体）
4. 激活 MyoD 基因家族，重编程为肌源性干细胞

# 共生进化

- 类型: 都市科幻
- 状态: 初始化
- 简介: 一个晚期癌症患者在非法生物实验中获得病毒-癌细胞共生能力，身体和大脑同时进化，在都市底层求生、追查真相、对抗阴谋的故事。
- 标签: 都市、进化、科幻、悬疑、底层逆袭
- 目标读者: 喜欢硬核科幻设定+都市生存+智斗的男性读者
- 预计字数: 200-300 万字
- 更新频率: 日更 4000-6000 字

## 核心卖点

1. **硬核科学设定**：病毒-癌细胞共生体系，所有能力都有生物学/神经科学依据
2. **能力代价机制**：每次使用能力都在消耗生命，停止运动就会退化甚至癌症复发
3. **底层逆袭爽感**：从绝症流浪汉到都市传奇，街头健身天花板+数学天才
4. **悬疑主线**：非法实验背后隐藏着更大的阴谋，主角逐步揭开真相

## 故事基调

- 前期：压抑、求生、底层挣扎
- 中期：觉醒、反击、能力成长
- 后期：对抗、抉择、终极进化

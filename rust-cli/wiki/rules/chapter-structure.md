# 章节结构模板

## 标准章节结构（2000-3000 字）

### 开头（200-400 字）
- **承接上文**：简要回顾前一章的关键信息
- **场景转换**：明确时间、地点变化
- **制造期待**：暗示本章将要发生的事

### 发展（800-1200 字）
- **推进剧情**：主角采取行动
- **制造冲突**：遇到困难或对手
- **展示成长**：主角能力或心智的提升
- **情感互动**：人物关系的深化

### 高潮（600-1000 字）
- **矛盾爆发**：冲突达到顶点
- **爽点呈现**：主角逆转或突破
- **情感共鸣**：让读者感受到主角的情绪

### 结尾（200-400 字）
- **收获总结**：主角获得什么（实力、宝物、人脉）
- **悬念设置**：新的问题或挑战出现
- **情感落点**：给读者一个情感出口

## 节奏控制
- **张弛有度**：高潮后要有缓冲，缓冲后要有新冲突
- **信息密度**：每章至少 3 个关键信息点
- **情绪曲线**：起伏波动，避免平铺直叙

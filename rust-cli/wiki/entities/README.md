# 角色实体目录

此目录用于存放从小说中提取的角色实体页面。

## 命名规范
- 文件名：角色名称.md（如：林凡.md、赵天虎.md）
- 格式：Markdown

## 页面结构
```markdown
# 角色名称

## 基本信息
- 姓名：
- 年龄：
- 身份：
- 修为：

## 性格特征
- 优点：
- 缺点：
- 特点：

## 能力设定
- 实力：
- 技能：
- 金手指：

## 人物关系
- 与主角的关系：
- 与其他角色的关系：

## 成长弧线
- 初始状态：
- 关键转折：
- 最终状态：
```

## 使用说明
1. 每创建一个新角色，在此目录下创建对应的 md 文件
2. 随着剧情发展，及时更新角色状态
3. 使用 `/wiki-health` 命令检查角色页面的完整性
